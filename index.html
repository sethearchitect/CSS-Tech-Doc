Introduction

CSS is the language we use to style a Web page. It stands for Cascading Style Sheets and we use it to describe how HTML appears on screens, paper, or any other media.

CSS is used to define

  • The design of your web pages
  • The layout of your pages and
  • The various displays for different screen sizes and devices

Why is CSS important?

Before CSS came, the styling of web pages were done in the HTML file and this could get very tedious and messy, especially as your project becomes bigger over time

CSS introduced division of labor and provided a way to remove the style formatting from the HTML file, so your HTML can simply focus on just defining the structure of your web pages. As most stylesheets are typically saved in external .css files, we are now able to keep our HTML docs clean, sweet, and easy to read.

CSS also gives you the power to change the whole look of an entire website by changing just one file!

Syntax

h1

{color: blue; text-align: center;}

The CSS syntax

A CSS rule is made up of the following syntax and parts

  • a selector
  • a declaration
  • properties
  • values

The selector is a like a hook that points and attaches to the HTML element you want to style

A declaration is a combination of a property and a value for that property, seperated by a colon [:]. when we have more than one declaration passed on a selector, its called a declaration block. A declaration block can have as many declarations surrounded by curly braces [{}]. In a declaration block, each of the declarations must end with a semi-colon [;] before another declaration starts.

Example

A selector with a single declaration

p {color: red;}

A selector with a declaration block:

p {
color: red;
font-size: 2px;
font-family: 'Arial', sans-serif;
}
  • p is a CSS selector that points to the element in the HTML file you want to style
  • color is a CSS property, and red is the CSS value we want that property to have
  • font-size is a CSS property as well, 2px is a CSS value
  • font-family is a CSS property, 'Arial', sans-serif are the CSS values we want this property to have. You will learn more about what's happening here in the coming sections.

CSS Selectors

CSS selectors are used to "find" or hook unto the HTML elements you want to style (edit). CSS selectors are grouped into five categories:

  • Simple selectors (select elements based on name, id, class)

    The CSS element selector

    The element selector selects HTML elements based on the element name.

    Example
    p {
    text-align: center;
    color: red;
    }

    Here, all < p> elements on the page will be center-aligned, with a red text color

    The CSS id selector

    The id selector uses the id attribute of an HTML element to select a specific element.

    The id of an element is unique within a page, so the id selector is used to select one unique element!

    To select an element with a specific id, write a hash (#) character, followed by the id of the element.

    Example
    #paragraph1 {
    text-align: center;
    color: red;
    }

    This CSS rule will be applied to the HTML element with id="para1":

    The CSS class selector

    The class selector selects HTML elements with a specific class attribute.

    To select elements with a specific class, write a period (.) character, followed by the class name.

    Example
    .bugger {
    text-align: center;
    color: red;
    }

    The HTML element(s) with class="bugger" will be red and center aligned

    More than one class can refer to a HTML element.

    Example
    < p class="center large" /> This paragraph refers to two classes.< /p>

    In this example the < p> element will be styled according to class="center" and to class="large"

    You can also specify that only specific HTML elements should be affected by a class.

    Example
    p.bugger {
    text-align: center;
    }

    < p> element(s) with class="bugger" will be center aligned

    • Other categories of selectors include:

  • Combinator selectors (select elements based on a specific relationship between them)
  • Pseudo-class Selectors (select elements based on a certain state)
  • Pseudo-element Selectors (select and style a part of an element)
  • Attribute Selectors (select elements based on an attribute or attribute value)

Adding CSS To Pages

We use stylesheets to direct browsers on how to display HTML documents. But since we mentioned earlier that CSS helped us seperate styles from the structure of our Pages the question becomes how do we connect our HTML doc with our stylesheets?

There are three ways to add CSS to your HTML

  • External CSS

    With an external style sheet, you can change the look of an entire website by changing just one file!

    Each HTML page must include a reference to the external style sheet file using the < link> element, inside the head section.

    Example

    External styles are defined within the < link> element, inside the < head> section of an HTML page:


    < html>
    < head>
    < link rel="stylesheet" href="mystyle.css">
    < /head>
    < body>
    < h1>This is a heading< /h1>
    < p>This is a paragraph.< /p>
    < /body>
    < /html>

    An external style sheet can be written in any text editor, and must be saved with a .css extension.

    The external .css file should not contain any HTML tags.

    Here is how the "mystyle.css" file looks:

    mystyle.css
    body {
    background-color: lightblue;
    }

    h1 {
    color: navy;
    margin-left: 20px;
    }

  • Internal CSS

    An internal style sheet may be used if one single HTML page has a unique style.

    The internal style is defined inside the < style> element, inside the head section.

    Example

    Internal styles are defined within the < style> element, inside the < head> section of an HTML page:


    < html>
    < head>
    < style>
    body {
    background-color: linen;
    }
    h1 {
    color: maroon;
    margin-left: 40px;
    }
    < /style>
    < /head>
    < body>
    < h1>This is a heading< /h1>
    < p>This is a paragraph.< /p>
    < /body>
    < /html>

  • Inline CSS

    An inline style may be used to apply a unique style for a single element.

    To use inline styles, add the style attribute to the relevant element. The style attribute can contain any CSS property.

    Example

    Inline styles are defined within the "style" attribute of the relevant element:


    < html>
    < body>
    < h1 style="color:blue;text-align:center;">This is a heading< /h1>
    < p style="color:red;">This is a paragraph.< /p>
    < /body>
    < /html>

CSS Functions

CSS has several useful functions that give you access to more functionality in styling your pages

CSS functions can be passed as values for various CSS properties. one of the most popular CSS function is the var() funtion

CSS Variables

The var() function is used to insert the value of a CSS variable.

You can create variables with local or global scope, change the variables with JavaScript, and change the variables based on media queries.

A good way to use CSS variables is when it comes to the colors of your design. Instead of copy and paste the same colors over and over again, you can place them in variables.

The syntax of the var() function is as follows:

var(--name, value)

The name must begin with two dashes (--) and is case sensitive

name This is required and defines the variable name. (Must start with two dashes)
values This is optional and is the fallback value to be used if the variable is not found

How var() works

First of all: CSS variables can have a global or local scope.

Global variables can be accessed/used through the entire document, while local variables can be used only inside the selector where it is declared.

To create a variable with global scope, declare it inside the :root selector. The :root selector matches the document's root element.

To create a variable with local scope, declare it inside the selector that is going to use it.

Example
:root {
--blue: #1e90ff;
--white: #ffffff;
}
body { background-color: var(--blue); }
h2 { border-bottom: 2px solid var(--blue); }
.container {
color: var(--blue);
background-color: var(--white);
padding: 15px;
}
button {
background-color: var(--white);
color: var(--blue);
border: 1px solid var(--blue);
padding: 5px;
}

Advantages of using var()

  • Makes the code easier to read
  • Makes it much easier to change the property values
